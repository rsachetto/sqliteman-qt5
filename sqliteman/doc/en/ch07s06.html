View related dialogs

There is the list of required properties you should fill:

Database

Select a database schema from this combo box.

View Name

Mandatory and unique (related to the Database) view name.

Select Statement

A valid SQL statement. Please do not write any CREATE or ALTER like commands. Your statement fails if you will do so.

You can create a view directly from SQL Editor - just use proper icon for chosen SQL statement.
