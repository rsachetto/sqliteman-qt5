Create Index

There is the list of required properties you should fill:

Index Name

You have to specify an unique index name.

Unique?

Check it if you want to have an unique index. Default is not unique.

Use?

Check it if you want to have this column contained in index.

Asc/Desc

Specify the index sort direction. Default is "ASC".
