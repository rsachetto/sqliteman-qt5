Schema Browser

All database objects are listed in this widget. Objects are grouped into logical tree. Available actions for chosen tree item (table, index, view etc.) are dynamically shown in the Context Menu in the menu bar or in the pop-up menu (right mouse click).

Pragma Editor

The database parameters can be changed comfortably in the Pragma Editor. All available PRAGMAs are listed in the table. Currently selected PRAGMA can be changed in the dialog opened with the Change Value....

Warning

Every PRAGMA change affects the behavior of the database. You should know what are you doing here.
