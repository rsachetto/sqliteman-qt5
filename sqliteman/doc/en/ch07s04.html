Import Table Data

Data in supported formats can be imported into chosen table directly in this dialog.

Schema to Import

Chose one of available databases. "Main" is the default. All attached databases are available.

Table to Import Into

Select the table for import. Only tables from chosen database are shown.

File to Import

A file from filesystem.

Table to Import Into

Select the table for import. Only tables from chosen database are shown.

Sqlite .import

This type of import handles plaintext files with separator as does the sqlite3 CLI console. Specify one of predefined field separators or create a custom one.

MS Excel XML

This type of import handles special XML files used in MS Excel. This tool cannot import any other XML file. This import filter does not contain any options.

Preview

Live preview of the selected file. You can check the result of import in the window.
