Requirements

In order to successfully use Sqliteman, you need Qt4 library at least 4.2. With Sqlite support (QSql module).

Important

It's strongly recommended to use later versions of Qt4 because there are some serious bugs fixed. Unfortunately some Linux distribution does not contain latest Qt libraries…

The Sqlite3 binary client is required for some actions. Sqliteman checks for its presence every start.

Packages for MS Windows are prepared with all required tools and libraries to use. Linux packages should provide requirements by distribution package system.
